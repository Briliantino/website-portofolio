# website-portofolio
Repo ini menjadi repo-master untuk mengumpulkan tugas besar mata kuliah Pemrograman Web Dinamis (PWD) sebelum UTS, yakni membuat website front-end portofolio.
Semua kelas harus membuat tugas tersebut, sebagai intrumen penilaian dengan bobot terbesar untuk nilai UTS. Tugas tersebut menjadi rangkaian tugas besar hingga UAS nanti, yakni membuat back-end dari website front-ent yang sudah dibuat.
